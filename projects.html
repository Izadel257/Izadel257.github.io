  • Git Conflict Daemon

    This is a lightweight, real-time Git conflict detector that runs quietly in the background and gently lets you know before you make changes to files that would result in merge conflicts.

    It is built with Python and uses the GitPython library to monitor changes in your Git repository. It offers the user the option to ignore the alert and continue working, or pull the latest changes from the remote repository.

  • Election News Data Analysis Project (Data Scientist, Carleton College)
    The Election News Analysis project has been angoing for more than 3 years and I was lucky to be one of the people that contributed to it. The project aims to collect transcripts from a range of news sources and analyze the trends and patterns in topics discussed during presidential election periods, as well as during major events that influenced those elections (e.g., the murder of George Floyd, the January 6th Capitol attack). So far, my contribution has been developing end-to-end data pipelines using Python, R, and SQL to extract, transform, and load over two years of news transcripts from NDS databases. I applied advanced data manipulation techniques with Pandas and NumPy to ensure the analysis aligned with project goals, ultimately supporting the processing of more than 60,000 transcripts. I also managed communications between Carleton faculty and NDS tech support, facilitating seamless collaboration across technical and academic teams. My role included leading data cleaning and transformation efforts, optimizing datasets for efficient and accessible use by end-users. Check out a blog about the project (not the most update but it should give you an idea of what the project is): Election News Data Analysis.
